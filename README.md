# vue-webpack

> 一个手工打造的vue开发脚手架  

> 初衷：[知乎上的一个问题](https://www.zhihu.com/question/271622369 "知乎上的一个问题")

# 快速开始

```bash
# 如果你用npm
npm install && cd server && npm install && cd ..

# 如果你用yarn
yarn install && cd server && yarn install && cd ..

#启动
npm start

```
